# Networking Basics for SOC Analyst

## Topics Covered
- LAN (Local Area Network)
- IP Addresses
- MAC Addresses
- Network Topologies

## Key Concepts

### IP Address
Identifies a device on a network. Used to track communication between systems.

### MAC Address
Unique hardware identifier assigned to a network interface.

### LAN
A local network connecting devices in a limited area (home, office).

### Network Topologies
Different ways devices are connected:
- Star
- Bus
- Ring

## SOC Relevance
SOC analysts use networking knowledge to:
- Identify suspicious IP activity
- Detect unusual connections
- Investigate lateral movement inside networks

## Key Takeaway
Understanding how devices communicate is essential for detecting abnormal or malicious traffic.
