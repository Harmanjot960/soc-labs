# TryHackMe - Introduction to Offensive Security

## Overview
This room introduced the basics of offensive security and ethical hacking.

## What I Learned
- Offensive security focuses on attacking systems to find vulnerabilities
- Ethical hackers simulate real-world attacks
- Capture The Flag (CTF) challenges are used for practice

## Key Concepts
- Exploitation of vulnerabilities
- Attacker mindset
- Importance of security testing

## SOC Relevance
Understanding attacker techniques helps SOC analysts:
- Recognize attack patterns
- Detect malicious behavior early
- Respond effectively to threats

## Key Takeaway
To defend systems effectively, SOC analysts must understand how attackers think and operate.
