# Mini Incident: Suspicious Network Activity

## Scenario
Unusual traffic detected from an internal device communicating with multiple unknown IP addresses.

## Investigation
- Reviewed IP activity
- Checked frequency of connections
- Observed multiple outbound requests

## Possible Issue
- Device may be scanning external systems
- Potential malware or unauthorized activity

## SOC Action
- Flagged device for monitoring
- Recommended further investigation

## Key Learning
Understanding IP behavior helps detect abnormal communication patterns early.
